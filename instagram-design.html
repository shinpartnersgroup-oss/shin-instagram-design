1意味の設計 / Meaning Design

存在の意味を定義する

ブランドとは「存在の意味」。意味を意図的に設計し、顧客が自発的に判断する仕組みを作る。意味だけでは人は動かない。社会課題と結びつけて「必要性」を生むことが必須。

Brand Meaning = Discovery
冷静に問題の本質を見抜き、
「そう来たか」と思わせる奇想天外な方法で
解決策を生み出し、みんなに発見をもたらす
一言キーワード: 発見 — 共感だけでは人は買わない。「発見がある人」として脳にアーカイブされることが購買の決め手。
01

あるべき姿（ゴール）

すべてのビジネスパーソンがAIを"自分の手"として使いこなし、自分の頭で考え、自分の力で道を切り拓ける社会

02

社会課題の本質

AIの波に飲まれて「思考停止」する人が急増。情報は溢れているのに何をすべきかわからない。AIに使われる側になっている

03

ターゲットの感情

誰かの役に立ちながら成長したい。自分らしくAIと向き合いたい。でも何から始めればいいかわからない

04

特異性 + 裏付け

圧倒的冷静さ × 言語化力 × 奇想天外な問題解決 × 抜けてる人間味。出来事ベースで証明する

2構造設計 / Structure Design

引き売り導線とルールの設計

Instagramでは一切売らない。ブランドの引力だけで、2つのビジネスに顧客が自発的に辿り着く構造を設計する。コンテンツの比率とフォーマットのルールで再現性を担保する。

発見のレイヤー構造 — 深さで自然に分岐する導線

設計思想: 各レイヤーが「発見の深さ」で自然に分かれる。押し売りゼロ。美化タイプは「もっと深く知りたい」という自分の感情に従って次のレイヤーへ進む。どのレイヤーにいても価値を感じられる。

Layer 0: Instagram（毎日）

発見の"入口"だけ見せる。「そう来たか」を1つ届ける。答えは全部言わない — 気になる状態を作る。

売り込み一切なし — ブランド構築のみ

Layer 1: YouTube（週1-2本）

Instagramで見せた「入口」の"答え"を10-15分で深掘り。動画で人柄が伝わり、美化タイプの信頼が加速する。

Instagram → YouTube の自然な流れ

Layer 1.5: LINE公式（任意 — なくても機能する）

入れる場合: 特典PDF + 週1配信で信頼構築。外す場合: YouTubeから直接サブスクへ。サブスクの無料ゾーン（40本）がLINE特典の代替になる。

状況に応じて導入/省略

Layer 2: AIサブスク（月額）

「もっと深い発見」が100本以上溜まる場所。40%無料 / 60%有料のフリーミアム。ランク制度（ブロンズ→シルバー→ゴールド）で「中途半端の不快」を活用し自発的課金を促す。

遠藤式「究極の引き売り」+ ゲーミフィケーション

Layer 3: 法人AI導入・研修（自然発生）

Layer 0-2のどこかで経営者が見つける。「この人にうちの会社を見てほしい」と自発的にDM or 問い合わせ。営業不要。

ブランドの引力で受注 — 全レイヤーが入口になる

レイヤー間の接続ルール

Instagram → YouTube

  • リールの最後: 「この続きはYouTubeで深掘りしてる。プロフから」
  • カルーセル最終枚: 「もっと詳しく知りたい人はYouTubeで」
  • ストーリーズ: YouTube公開時に「今日の動画、これはヤバい」とチラ見せ

YouTube → サブスク

  • 動画内: 「これの完全版はサブスクにある」（月1-2回だけ。毎回は言わない）
  • 概要欄: サブスクのリンクを控えめに設置
  • 無料ゾーン40本がリードマグネット代わり: 「登録するだけで40本無料」

全レイヤー → 法人

  • 導線を作らない。ブランドの引力だけで来させる
  • 本質解説系の投稿が経営者に刺さる
  • DM or プロフのリンクに問い合わせ先を控えめに設置

コンテンツ4カテゴリと比率ルール

40%

発見系 — 「そう来たか」

  • みんなChatGPTに"質問"してる。僕は"議論"させてる。
  • AI時代に最も価値が上がるスキルは、プログラミングじゃない。
  • この業務、普通なら3人で1週間。僕はAIと2時間で終わらせた。
  • "AIに仕事を奪われる"は嘘。本当に奪われるのは○○。
30%

生き様系 — 模倣不可能

  • クライアントの問題を解決した過程（結果ではなくプロセス）
  • AI効率化のプロが手書きでメモを取っている瞬間
  • 道に迷った話、予定を間違えた話
  • 「今日クライアントにこう言われた。嬉しかった。」
20%

本質解説系 — 冷静な言語化

  • AI導入が失敗する会社の共通点。ツールの問題じゃない。
  • "AIは危険"という人が見落としている本質
  • 情報過多の時代に、なぜ人は思考停止するのか
10%

与える系 — 見返りゼロ

  • 経営者が今日から使えるAIプロンプト5選（保存推奨）
  • AI初心者がまずやるべき3ステップ
  • 僕が毎日使ってるAIワークフロー、全部見せます

遠藤式「究極の引き売り」4つの構造

STRUCTURE 01

発見の入口だけ見せる

リールで3つのうち1つだけ教え、残りはサブスク内で解説。

「AIで業務を自動化する方法、3つあるんだけど... 1つだけ教える」
STRUCTURE 02

裏側をチラ見せ

ストーリーズで「収録した」とだけ伝え、中身は見せない。違和感を作る。

「今日サブスクメンバー向けに収録した内容、これはヤバい。」
STRUCTURE 03

無料の質で圧倒する

無料コンテンツの質を異常に高くし、「有料はどうなってるんだ」と思わせる。

「僕が毎日使ってるAIワークフロー、全部見せます（保存推奨）」
STRUCTURE 04

メンバーの変化を事実共有

勧誘ではなく事実。「○○さんがこんな成果を出した」。

「サブスクメンバーの方が、AI導入で月30時間の削減に成功した話。」
3感情設計 / Emotion Design

タッチポイントごとの感情を設計する

美化タイプがInstagramで北山伸に出会い、ファンになり、サービスに辿り着くまでの「感情の旅」を設計する。各段階で狙う感情を明確にし、逆算してコンテンツを作る。

Phase 1 — 初接点

「なんか違う人がいる」

リールやカルーセルで初めて出会う。冷たいテック感ではなく温かいのに鋭い。煽りもない。この違和感で指が止まる。

好奇心 — 「この人、他のAI発信者と何か違う」
Phase 2 — フォロー直後

「発見の連続だ」

投稿を遡ると、毎回「そう来たか」がある。共感もあるが、それ以上に発見がある。脳が「この人=発見」とアーカイブする。

興奮 — 「AIってそういう使い方あるんだ」
Phase 3 — 信頼構築

「この人、本物だ」

生き様系の投稿で人間性が見える。AI効率化のプロが手間を大事にする。失敗もさらけ出す。営利色がない。この人は本気で人の役に立とうとしてる。

信頼 — 「この人についていきたい」
Phase 4 — 欲求発生

「もっと深く知りたい」

無料コンテンツの質が高すぎて、有料がどうなってるのか気になる。サブスクメンバーの変化を見て「自分も変わりたい」。自発的にリンクをタップする。

渇望 — 「この人からもっと学びたい」
Phase 5 — 推奨

「この人、みんなに知ってほしい」

美化タイプは「人のために」シェアする。良い発見を独り占めせず、大切な人に教えたくなる。口コミが自然発生する。

使命感 — 「私が見つけた発見を届けたい」

美化タイプの感情トリガー

  • 「与えてくれている」と感じた瞬間 → 信頼が跳ね上がる
  • 「この人も完璧じゃない」と感じた瞬間 → 親近感と安心
  • 「言語化してくれた」と感じた瞬間 → 自分のモヤモヤが晴れる快感
  • 「自分もこうなりたい」と感じた瞬間 → 購買の入口
  • 「この人を応援したい」と感じた瞬間 → シェア・推奨の入口

感情を壊す地雷（美化タイプ特有）

  • 営利色を感じた瞬間 → 「結局お金か」で一瞬で離脱
  • 言行不一致を感じた瞬間 → 「嘘だったんだ」で信頼崩壊
  • マウントを感じた瞬間 → 「この人、自分のためにやってる」で冷める
  • 煽りを感じた瞬間 → 「またこのパターンか」でマーケ疲れ
  • 他者批判を見た瞬間 → 美化タイプは争いを最も嫌う
4空気設計 / Atmosphere Design

全タッチポイントの空気を統一する

非言語で意味を伝える。「AIなのに温かい」というギャップそのものが発見になる。カラー、写真、フォント、語り口、BGM — すべてが同じ空気を纏う。

カラーパレット

ベージュ
オフホワイト
セージ
モスグリーン
チャコール

温かみのあるナチュラルトーン。美化タイプが好む柔らかさと、「AI発信なのにテック感がない」違和感を両立。冷たいブルー系・ネオン系は排除。

写真・動画のトーン

  • 光: 自然光、柔らかい陰影。蛍光灯・オフィス感はNG
  • 構図: 作業する手元、ホワイトボード、カフェの思考シーン
  • 空気: 「考えている人」の静かな熱量
  • 顔出し: する。表情は穏やか、目に芯がある
  • NG: ブランド品・高級感の誇示、オフィスの権威演出

語り口のトーン設計

  • 通常の投稿 「これ、意外とみんな気づいてないんだけど...」
  • 核心を突くとき — タメ口で言い切り 「それ、AIの問題じゃない。考えてないだけ。」
  • 失敗談・抜けてる話 「また道間違えた。AI使ってるのにGoogleマップは使えない」
  • 発見を届けるとき — 静かな熱量 「ここだけの話、これ気づいてる人ほぼいない。」

リールBGM・テンポ

  • BGM: 落ち着いたアコースティック / Lo-Fi系。EDM・煽り系は一切使わない
  • テンポ: 速すぎず、でもダレない。呼吸がある
  • テロップ: 丸みのあるフォント、余白多め、1画面に情報を詰めない
  • SE: 過剰な効果音なし。あっても控えめに

行動タイプ向けの「刺激的で速いリール」とは真逆。美化タイプが心地よく「発見」に浸れる空気を作る。

5メディア設計 / Media Design

具体的な投稿フォーマットと運用

意味・構造・感情・空気の設計を、実際の投稿に落とし込む。リール・カルーセル・ストーリーズの3本柱で、週間スケジュールに沿って運用する。

追わない指標（罠になる）

  • リーチ数 — 初期は少なくて当然。ブランドの深さを優先

鉄則: やること / やらないこと

✓ 絶対にやること

  • 人に与える姿勢を行動で見せる
  • 手間のかかることを大事にする姿を見せる
  • 失敗や抜けてる部分も正直に出す
  • クライアントの成果を誇る（自分の成果ではなく）
  • 学び続ける姿勢を見せる
  • 共感より「発見」を常に意識する
  • 非言語（写真・動画・行動）で意味を伝える
  • CTA は「届けたい人がいたらシェアして」

✗ 絶対にやらないこと

  • 年収・年商・売上の開示
  • 「○○万円稼いだ方法」系の投稿
  • 派手なライフスタイルの誇示
  • 「期間限定」「残りわずか」等の煽り
  • 他者批判・マウント
  • 毎日の売り込みストーリーズ
  • CURAIMの名前を出す
  • 一貫性のない発信（言行不一致は即アウト）
